# Celesti Craft Website

Single-page HTML website for **Celesti Craft** (forged speaker hardware manufacturer, Ghaziabad).

## Current file

- `celesti-craft-v3 (2).html` (main website file)

## Recommended publish filename

For GitHub Pages and most static hosts, rename the page to:

- `index.html`

## Publish to GitHub (quick steps)

1. Create an empty repository on GitHub (for example: `celesti-craft-website`).
2. In this folder, run:

```bash
git add .
git commit -m "Initial Celesti Craft website"
git branch -M main
git remote add origin https://github.com/<your-username>/celesti-craft-website.git
git push -u origin main
```

## Optional: Enable GitHub Pages

1. In GitHub repo: **Settings -> Pages**
2. Source: **Deploy from a branch**
3. Branch: **main** and folder: **/(root)**
4. Save

Your site will be live at:

- `https://<your-username>.github.io/celesti-craft-website/`
