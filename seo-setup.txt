Celesti Craft — SEO & discovery setup (forgedspeakerkit.in)
=========================================================

1) Google Search Console
   - Open https://search.google.com/search-console
   - Add property: https://forgedspeakerkit.in/
   - Verify using the HTML meta tag method.
   - Google Search Console HTML meta tag is already on index.html (home); keep it after verification.
   - After verification: Sitemaps → submit exactly this URL (no trailing slash after .xml):
     https://forgedspeakerkit.in/sitemap.xml
     If you use https://forgedspeakerkit.in/sitemap.xml/ (slash at end), GitHub Pages returns HTML 404 and Search Console shows “could not be read”.
   - Open that URL in a browser — you should see raw XML starting with <?xml version=
   - Monitor Coverage / Page indexing for errors; request indexing for important URLs after large changes.

2) Bing Webmaster Tools
   - Import from Google Search Console, or verify with meta tag (msvalidate.01) and add it next to the Google verification comment in index.html.
   - Submit the same sitemap URL.

3) Google Analytics 4
   - The official Google tag (gtag.js) for Measurement ID G-KZXLMT4GW3 is in <head> on site templates (see index.html, contact, products, etc.).
   - To change the property ID later: replace G-KZXLMT4GW3 in each <!-- Google tag (gtag.js) --> block (same ID everywhere — only one tag per page).
   - Optional: assets/js/analytics.js sends extra events (WhatsApp, phone, PDF, forms); keep loading it after the Google tag (defer at end of body is fine).

4) Google Business Profile (Maps)
   - Ensure business name, address, phones match NAP.txt exactly where possible.
   - Primary category: manufacturing / industrial supplier aligned with your offering.
   - Website URL: https://forgedspeakerkit.in/
   - Add links to Contact, Products, and PDF catalogue where GBP allows (website + appointment/posts).

5) GitHub Pages vs custom domain
   - See HOSTING_REDIRECTS.txt for consolidating traffic on forgedspeakerkit.in.

6) Citations and backlinks (match NAP.txt everywhere)
   - Claim selective listings on India B2B / industrial directories buyers actually search.
   - Keep identical business name, address, phones, email, and https://forgedspeakerkit.in/.
   - Ask steady customers for a short testimonial page linking to your site when appropriate.
   - Trade fairs / association profiles: link using the canonical domain only.
